# Football Analytics Platform

End-to-end football analytics project built with Python, Pandas, scikit-learn, ClickHouse and Superset.

## Goals
- Match outcome prediction
- Goals-related prediction
- Team form analytics
- Player performance modeling
- Tournament and World Cup simulation

## Stack
- Python 3.12
- Pandas
- scikit-learn
- ClickHouse
- Superset

## Project structure
- `data/` - datasets
- `notebooks/` - exploratory analysis
- `src/` - source code
- `sql/` - database scripts
- `dashboards/` - dashboard notes and screenshots
- `models/` - trained model artifacts
- `reports/` - outputs and reports
